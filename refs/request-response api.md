# request-response api
#### status: 
#### created: 2022-07-21

the request response is a design pattern where the requestor send a request using *http* methods to a resource on the internet which lives in a web sever which on his turn will return a response object (Json or XMl) and so on . for instance there is many apis paradigm that use this pattern, one of them is the **REST** paradigm
